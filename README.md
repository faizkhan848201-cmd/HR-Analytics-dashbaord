HR Analytics Dashboard – Data-Driven Employee Attrition Insights (Power BI)

A Power BI dashboard delivering actionable insights into employee attrition across demographics, roles, tenure, and salary—designed to support HR decision-making and workforce planning.

Project Overview :

This project presents an HR Analytics Dashboard built using Power BI to analyze and visualize employee attrition. It provides a clear view of key HR metrics such as:

Attrition Count
Attrition Percentage
Total Employee (FTE) Count
Average Salary
Average Age
Average Years of Experience
Gender-wise Attrition

The dashboard also highlights attrition patterns across multiple dimensions, including:

Age Group
Education Background
Salary Slab
Years at Company
Job Role

These insights help HR teams identify risk areas, understand workforce behavior, and build better employee retention strategies.
